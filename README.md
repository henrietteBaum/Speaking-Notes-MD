# Speaking-Notes-Markdown

Demo-Project for learning purpose.
Provide accessibility for visual-impared users.

Simple markdown-editor uses React.js, React-markdown, Styled-Components, Context-API.

## Dependencies
- node.js
- create-react-app
- react-router-dom
- styled-components
- bootstrap
- react-bootstrap
- react-markdown
